SVU is a tilde community mostly consisting of users who share a unix based operating system (currently Ubuntu 22.04 LTS). We talk, socialize and self host services which we find useful.

Welcome!!
Each user will get a home directory with a few files already existing. A user can use gameshell to learn the basic of bash. There's also seperate lvms mounted where you can store backups, add your own music, movies and shows.
