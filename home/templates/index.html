Each user will get a home directory with a few files already existing. A user can play gameshell.sh to learn the basic of bash. There's also seperate lvms mounted where you can store backups, add your own music, movies and shows. A few things to remember:

  • We do not have disk quotas so please be mindful of how much you upload to the server
  • You'll have to be connected to tailscale to access svu
  • You can request any changes or services you would like through iris

What is a Tilde Community?



A tilde community is essentially a collection of users sharing a *nix system. Members of the community use the system as a platform to collobarate, build cool things and socialize with each other. tilde.club was the first tilde community that introduced the idea of a shared nix system after which several have originated, forming the tildverse
